# GPT_clone
## Project Overview

This repository contains two main parts:

1. **GPT from Scratch**  
   The `GPT_pretrained` notebook/script builds a GPT-style model from scratch, covering the core architecture and workflow used in transformer-based language models.

2. **Spam Classification with Fine-Tuning**  
   The classification part uses the pretrained GPT model and fine-tunes it for a **spam classification** task.
