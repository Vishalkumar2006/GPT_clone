# GPT_clone
## About the Project

This project explores the implementation and application of GPT models in two stages:

- **Building GPT from scratch:**  
  The `GPT_pretrained` component develops a GPT-style language model from the ground up to understand its architecture and working principles.

- **Fine-tuning for spam classification:**  
  The pretrained model is then adapted and fine-tuned for a **Spam Classification** task.
